
# Overview

`scene-gazer` is a video renderer, that is able to load video metadata.

## Example: Metadata File

```
{
    "tags": [
        "Cateogry / Label"
    ],
    "chapters": [
        {
            "description": "La-bla!",
            "timestamp": "00:00:00.000"
        }
    ],
    "highlights": [
        {
            "description": "La-bla!",
            "timestamp": "00:00:00.000"
        }
    ]
}
```
